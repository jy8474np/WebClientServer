HELLO INPUT : The Program That Started It All

It seems like only last week that I wrote this program. THE program, really.

Hello Input was more than a program. What first appears to be simple, intro-level program introducing a new programmer to the mechanics of creating various types of inputs, drop down menus, and action buttons eventually becomes an exercise in existential examination.

  • In 1595 Willam Shakespeare asked the world, "What's in a name?" Hello Input demands an answer to that question.
  • Hello Input requires a password; obviously symbolizing the war between humankind's simultaneously duplicitous and trusting natures.
  • In a nod to the 1979 novel and subsequent 1982 film, Sophie’s Choice , Hello Input asks you to choose just one of your favorite program languages.
  • Finally, Hello Input offers a scathing critique of the human cost of modernity's Need It Now delivery methods.
